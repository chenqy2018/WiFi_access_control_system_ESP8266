#WiFi无线门禁考勤系统_WiFi模块

----------------------------------------------------------------------

## 1. 系统介绍

​	本系统采用stm32f103zet6作为控制器，可以刷卡、刷指纹验证用户身份，通过ESP8266 Mesh组网与服务器通讯。

​	服务器端可以添加、删除、修改人员信息，读取设备考勤信息，并输出excel表格。
